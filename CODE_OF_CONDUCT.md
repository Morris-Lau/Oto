# Code of Conduct

## Our Standard

We want this project to be a respectful, constructive place to collaborate.

Examples of expected behavior:

- be kind and precise in technical discussion
- focus feedback on the work, not the person
- assume good intent while still being honest and direct
- help newcomers get oriented

Examples of unacceptable behavior:

- harassment, personal attacks, or intimidation
- discriminatory language
- repeated bad-faith arguing
- doxxing or sharing private information without consent

## Enforcement

Project maintainers may edit, remove, or reject contributions and discussions that do not meet this standard.

## Reporting

If someone violates this code of conduct, contact the maintainer privately when possible instead of escalating the conflict in public.
